                          Text&Window

                          Text元素

                          Text - 在窗口中显示一些文本。通常是指一行文本。不过，文本也可以是多行。如果是多行，则没有滚动条。

                          以下是Text在PySimpleGui中的函数定义：

                          class Text(Element):
    def __init__(self, text='', size=(None, None), s=(None, None), auto_size_text=None, click_submits=False, enable_events=False,
    relief=None, font=None, text_color=None, background_color=None, border_width=None, justification=None, pad=None, p=None, key=None,
    k=None, right_click_menu=None, expand_x=False, expand_y=False, grab=None, tooltip=None, visible=True, metadata=None):

    # text： 要显示的文本。可包含 /n 以实现多行显示。 将参数（可选）转换为字符串
    # size: (w, h) w=字宽，h=行高。如果提供的是一个 int 而不是一个 tuple，那么高度将自动设置为 1
    # s：与 size 参数相同。 这是一个别名。如果两个参数都被设置，则使用被设置的参数。如果两个都设置了，则使用 size 参数
    # :auto_size_text：如果设置为 True，文本元素的大小将根据 "text "参数中提供的字符串调整。
    # click_submits：请勿使用。仅列出用于向后兼容 - 使用 enable_events 代替
    # enable_events：打开特定元素事件。文本事件在点击文本时发生
    # relief：文本周围的浮雕样式。值与进度表浮雕值相同。应该是以 "RELIEF_"开头定义的常量 - RELIEF_RAISED、RELIEF_SUNKEN、RELIEF_FLAT、RELIEF_RIDGE、RELIEF_GROOVE、RELIEF_SOLID
    # font：指定字体系列、大小等。元组或单个字符串格式为 "name size styles"。样式：斜体 * 罗马体 粗体 正常下划线 粗体
    # text_color：文本颜色
    # background_color：背景颜色
    # border_width：边框的像素数（如果使用浮雕的话）
    # justification：字符串在 size 提供的空间内的对齐方式。有效选项 = `左`、`右`、`中`。
    # pad：元素周围的填充量，单位为像素（左/右，上/下）或（（左，右），（上，下））或一个 int。如果是 int，则转换为一个元组（int, int）
    # p：与 pad 参数相同。 这是一个别名。如果两个参数都被设置，则使用被设置的参数。如果两个都设置了，则使用 pad 参数
    # key：与 window.find_element 和返回值一起使用，用于唯一标识该元素，以唯一标识该元素
    # k：与 key 相同。您可以使用 k 或 key。无论设置了哪一个，都将被使用。
    # right_click_menu： 右键单击该元素时要显示的菜单项列表。具体格式请参见用户文档。
    # expand_x： 如果为 True，元素将自动向 X 方向展开以填充可用空间
    # expand_y： 如果为 True，元素将自动向 Y 方向扩展以填充可用空间
    # grab： 如果为 True，可以抓取该元素并移动窗口。默认为 False
    # tooltip：文本，当鼠标悬停在元素上时显示
    # visible：设置元素的可见状态
    # metadata：元数据： 用户元数据，可以设置为任何内容

                          Window元素

                          Window - 表示一个窗口

                          以下是Window在PySimpleGui中的函数定义：

                          class Window:
    def __init__(self, title, layout=None, default_element_size=None,
                 default_button_element_size=(None, None),
                 auto_size_text=None, auto_size_buttons=None, location=(None, None), relative_location=(None, None), size=(None, None),
                 element_padding=None, margins=(None, None), button_color=None, font=None,
                 progress_bar_color=(None, None), background_color=None, border_depth=None, auto_close=False,
                 auto_close_duration=DEFAULT_AUTOCLOSE_TIME, icon=None, force_toplevel=False,
                 alpha_channel=None, return_keyboard_events=False, use_default_focus=True, text_justification=None,
                 no_titlebar=False, grab_anywhere=False, grab_anywhere_using_control=True, keep_on_top=None, resizable=False, disable_close=False,
                 disable_minimize=False, right_click_menu=None, transparent_color=None, debugger_enabled=True,
                 right_click_menu_background_color=None, right_click_menu_text_color=None, right_click_menu_disabled_text_color=None,
                 right_click_menu_selected_colors=(None, None),
                 right_click_menu_font=None, right_click_menu_tearoff=False,
                 finalize=False, element_justification='left', ttk_theme=None, use_ttk_buttons=None, modal=False, enable_close_attempted_event=False,
                 titlebar_background_color=None, titlebar_text_color=None, titlebar_font=None, titlebar_icon=None,
                 use_custom_titlebar=None, scaling=None,
                 sbar_trough_color=None, sbar_background_color=None, sbar_arrow_color=None, sbar_width=None, sbar_arrow_width=None,
                 sbar_frame_color=None, sbar_relief=None,metadata=None):

    # title：标题：将显示在标题栏和任务栏上的标题
    # layout（布局窗口的布局。也可在布局方法中指定
    # default_element_size：该窗口中所有元素的大小，以字符（宽）和行（高）为单位
    # default_button_element_size：此窗口中所有按钮元素的大小（宽、高），单位为字符数（宽）和行（高
    # auto_size_text（自动调整文字大小如果窗口中元素的大小应与文本长度相等，则设置为True。
    # auto_size_buttons：如果此窗口中按钮的大小应与文本完全匹配，则设置为True。
    # relative_location：（x,y）相对于窗口默认位置的位置，单位为像素。通常情况下，窗口会居中。该位置相对于窗口创建时的位置。注意，它们可以是负数。
    # location：（x,y）窗口左上角在屏幕上的位置，单位为像素。默认为位于屏幕中央。
    # size：（宽度，高度）窗口的尺寸，单位为像素。通常情况下，窗口会根据内容自动调整大小，而不是由用户设置绝对大小。尽量不要设置此值。否则会有内容被截断等风险。让布局来决定窗口大小吧
    # element_padding：窗口中元素周围的默认填充量（左/右、上/下）或（（左、右）、（上、下）），或一个int。如果是int，则转换为一个元组（int,int）
    # margins：（左/右、上/下）在显示元素之前，在窗口边框内边缘留出的像素数。
    # button_color：窗口中所有按钮的默认按钮颜色
    # font：指定字体系列、大小等。元组或单个字符串格式为"名称大小样式"。样式：斜体*罗马体粗体正常下划线粗体
    # progress_bar_color：（条形图颜色、背景颜色）设置窗口中所有进度条的默认颜色
    # background_color：背景颜色
    # border_depth：边框深度：窗口中所有元素的默认边框深度（宽度
    # auto_close：自动关闭：如果为True，窗口将自动关闭
    # auto_close_duration：自动关闭持续时间：关闭窗口前的等待秒数
    # icon：可以是文件名或Base64值。对于Windows，如果是文件名，则必须是ICO格式。对于Linux，必须不是ICO格式。最便携的方法是使用PNG文件的Base64值。这适用于所有操作系统
    # force_toplevel：如果为True，将使该窗口跳过隐藏主窗口的正常使用
    # alpha_channel（阿尔法通道）：设置窗口的不透明度。0=不可见1=完全可见。0和1之间的值会在某些环境中产生半透明窗口（RaspberryPi始终将此值设为1，且无法更改。
    # return_keyboard_events：如果为True，键盘上的按键将作为读取调用的事件返回
    # use_default_focus：如果为True，将使用默认焦点算法将焦点设置为"正确"元素
    # text_justification：窗口中所有文本元素的默认文本对齐方式
    # :typetext_justification:'left'|'right'|'center
    # no_titlebar：如果为true，窗口上将不显示标题栏或框架。这意味着无法最小化窗口，窗口也不会显示在任务栏上
    # grab_anywhere：如果为True，则可以使用鼠标点击和拖动来移动窗口。除了某些系统上的输入框外，窗口的几乎所有位置都可以移动
    # grab_anywhere_using_control：如果设置为True，则可以使用CONTROL键+鼠标左键单击并拖动来移动窗口。默认值为TRUE。与普通的随处抓取不同，它适用于所有元素。
    # keep_on_top（保持在顶部）：如果为True，创建的窗口将位于屏幕上所有其他窗口的顶部。如果使用该参数创建了另一个窗口，则该窗口可能会被向下移动。
    # resizable：如果为True，则允许用户调整窗口大小。请注意，并非所有元素在调整大小时都会改变大小或位置。
    # disable_close（关闭）：如果为True，窗口右上角的X按钮将不起作用。请谨慎使用，并始终为用户留出退路。
    # disable_minimize：如果为True，用户将无法最小化窗口。有利于占据整个屏幕并保持这种状态。
    # right_click_menu：右键单击该元素时要显示的菜单项列表。具体格式请参见用户文档。
    # transparent_color（透明色）：窗口中带有此颜色的任何部分都将完全透明。您甚至可以通过这些位置点击该窗口下的窗口。
    # debugger_enabled（已启用调试器）：如果为True，则将启用内部调试器
    # right_click_menu_background_color:右键菜单的背景颜色
    # right_click_menu_text_color:右键菜单的文本颜色
    # right_click_menu_disabled_text_color:禁用右键菜单项的文本颜色
    # right_click_menu_selected_colors：选定项目的文本和背景颜色。可以是一个元组，也可以是一个颜色字符串。如果只想设置背景，可以是单一颜色。通常是一个元组，但也可以是简化的双色字符串"前景背景"。如果只想设置背景，可以是单一颜色。
    # right_click_menu_font：右键菜单的字体
    # right_click_menu_tearoff:如果为True，则所有右键菜单都可以撕掉
    # finalize（最终确定如果为True，将调用Finalize方法。使用此方法比使用.Finalize方法更简洁
    # element_justification：窗口本身中的所有元素都将具有该理由，"左"、"右"、"中"均为有效值
    # ttk_theme：设置窗口的tkinterttk"主题"。默认值=DEFAULT_TTK_THEME。将所有ttk部件默认设置为该主题
    # use_ttk_buttons：影响窗口中的所有按钮。True=使用ttk按钮。False=不使用ttk按钮。无=仅在Mac上使用ttk按钮
    # modal（模态）：如果为True，则该窗口将是用户在关闭前唯一可以与之交互的窗口
    # enable_close_attempted_event：如果为True，则点击"X"后窗口不会关闭。取而代之的是从window.read返回的事件WINDOW_CLOSE_ATTEMPTED_EVENT
    # titlebar_background_color:如果使用use_custom_titlebar指示自定义标题栏，则使用此颜色作为背景色
    # sbar_frame_color:滚动条周围边框的颜色（仅适用于某些ttk主题）
    # sbar_relief：用于滚动条"拇指"（滑动时抓取的部分）的滚动条浮雕。应该是以"RELIEF_"开头定义的常量-RELIEF_RAISED、RELIEF_SUNKEN、RELIEF_FLAT、RELIEF_RIDGE、RELIEF_GROOVE、RELIEF_SOLID
    # metadata：用户元数据，可设置为任何内容

                          Demo1.1：Hello World.py

                          通过使用PySimpleGui的Text和Window，以下程序构建了一个含有文本的简单窗口：

                          import PySimpleGUI as sg

Text = sg.Text('Hello World!') # 定义一个文本元素
layout = [[Text]] # 创建布局

window = sg.Window('Window Title', layout, size = (300, 50), element_justification = 'c') # 创建窗口

while True: # 事件循环并获取输入值
    event, values = window.read()
    if event in (None, 'Cancel'):   # 如果用户关闭窗口或点击`Cancel`
        break
window.close() # 跳出循环后关闭窗口

                          其中，import PySimpleGUI as sg引入PySimpleGUI包，并命名为sg，Text = sg.Text('Hello World!')定义了一个文本为“Hello World!”的文本元素， layout = [[Text]]创建了一个布局，并将先前定义的文本元素置入，window = sg.Window('Window Title', layout, size = (300, 50), element_justification = 'c')创建了一个标题为“Window Title”，布局为layout，大小为300x500px，元素居中对齐的窗口。

                          而后创建了一个循环以获取输入值，并定义了关闭窗口事件的判断语句，该语句可先简单了解，对PySimpleGui的事件暂时不作过多介绍。
